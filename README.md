# Combine UI, API Automation

Test automation for Business Portal.

## Execute Playwright tests locally

1) Make sure that you have already installed [NodeJS](https://nodejs.org/uk/download/) on your computer
2) Clone repository <https://bitbucket.org/veryfastio/playwright_tests/src/main>
3) Open terminal window and switch to directory: ./playwright_tests
4) Execute `npm install`
5) Make sure you have .env.local file with credentials(ask someone from AQA team for this file)
6) Execute `npx playwright install`
7) Execute the following commands:
   `npx playwright test --ui` in order to run all tests. More details - <https://playwright.dev/docs/running-tests>
